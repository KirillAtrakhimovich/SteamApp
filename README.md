The first view presents a list of games, which we can make favorite. 

If we pick the game we can see all information about the game.

### Third View :

The third view presents a list of news about our favorites games chronologically.

Tap on a row presents the details screen.

The Filter button presents the filter list over the current screen (blurred). Unchecked games are removed from the news list

The last screen displays the contents of each particular news record.
